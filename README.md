# Prueba-Tecnica-IZUMU-Clientes
Prueba Tecnica con frontend, backend y script de base de datos.

El script de base de datos esta en la carpeta IZUMU Clientes Base de datos. Se desarrollo para sql server y se hizo ejecuto localmente con la conexión sql EXPRESS.Ejecutarlo con las mismas condiciones para ver reflejada la creación.

Para ejecutar el backend abrir la carpeta IZUMU Clientes backend y despues la carpeta IZUMU Clientes, alli se encuentra la solución para abrila en Visual Studio. Una vez dentro de Visual Studio dar click derecho en la solución y hacer click en Clean y despues hacer click en Rebuild para que se restauren los packages necesarios. Al correr la solución, el backend queda expuesto en la siguiente url https://localhost:7190.


Para ejecutar el frontend abrir la carpeta IZUMU Clientes frontend alli se encuentra la solución para abrila en Visual Studio. No alcance a completar todo el flujo por lo que el front es solo una cascara y no ejecuta logica.
